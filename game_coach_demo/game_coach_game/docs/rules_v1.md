# 数字预言家规则文档 v1.0

## 1. 基础流程
- 两名玩家进入同一房间后开始对局。
- 棋盘为 `rows x cols`，玩家轮流在空格落子。
- 输入允许数字字符串（如 `1`、`5`）或 `X`。

## 2. 合法性规则
- 同一行内数字不能重复。
- 同一列内数字不能重复。
- `X` 不参与数字重复校验，可重复出现。
- 已填充格子不能再次落子（若使用改格能力，需符合改格规则）。

## 3. 计分规则（行/列满线触发）
- 只有某一行或某一列被填满时，才进行该线计分。
- 令 `n = 该线中非X数字个数`。
- 若该线上存在值等于 `n` 的格子，则该格子归属玩家获得 `n` 分（每个玩家在该线最多触发一次）。

## 4. 卡牌规则
- `double_score`：该玩家最终结算分数乘以 2。
- `skip_turn`：对手下一次回合被跳过（服务端状态生效）。
- `change_number`：允许修改自己已落子的格子，但修改后仍需通过合法性校验并重算受影响行/列分数。

## 5. 结束规则
- 棋盘填满后游戏自动结束。
- 最终结算分数为 `局内分数 * multiplier`。
- 总积分会累加最终分，同时更新胜负平记录。

## 6. 版本与兼容
- 当前版本：`RULES_VERSION = v1.0`
- 所有教练快照与评估接口应返回/依赖该版本字段，便于评测与回放一致性验证。
